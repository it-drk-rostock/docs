## August 2024

### Neue Funktionen

- Auf der Berechtigungsseite zum Hinzufügen von Benutzern zu Rollen kann jetzt die Mehrfachauswahl genutzt werden, um mehrere Benutzer gleichzeitig auszuwählen.
- Ein Filter für Benutzer wurde zur Berechtigungsseite hinzugefügt, um die Auswahl der Benutzer zu erleichtern.

### Bug Fixes

### Änderungen

- das Breadcrumb-Aktionssymbol: Variante und Farbe wurden angepasst
- ein Datumsbereich „Von-Bis“ wurde zur Exportfunktion hinzugefügt, um den gleichzeitigen Export mehrerer Tage zu ermöglichen
- Im Formular zum Erstellen von Turmtagen wurde der Filter angepasst, um nun auch nach Standort zu filtern. Außerdem wurde der Dropdown so aktualisiert, dass er eine maximale Höhe hat und scrollbar ist.
- Eine eindeutige Einschränkung für Turmtage wurde verändert, sodass nun nur noch ein Turmleiter pro Tag und pro Turmtag ausgewählt werden kann.
